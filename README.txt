# Imposteur — Web mobile (MVP)
## Installation
1) Installer Node.js (https://nodejs.org)
2) Ouvrir un terminal dans ce dossier et exécuter :
   npm init -y
   npm install express socket.io

## Démarrage
node server.js
→ Ouvrir http://localhost:3000 sur votre PC et/ou smartphone (même réseau Wi‑Fi).

## Jouer
- Un joueur clique « Créer », partage le code.
- Les autres « Rejoindre » avec le code.
- L’hôte peut coller une liste personnalisée (format `commun|imposteur` par ligne, ou un mot par ligne).
- Démarrer la manche -> Indices -> Vote -> Résultats.
